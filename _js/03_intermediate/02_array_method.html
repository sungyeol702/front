배열관련 메서드 : 유사(연관)배열에서는 사용제한이 있음

  • join() : 배열원소 사이사이에 문자를 삽입하여 새로운 문자열을 만들어주는 메서드
  • array.push(value) : 배열의 마지막에 원소를 추가 : array[array.length] = value 과 같다
    추가 후 배열의 길이 리턴
  • pop() : 배열의 마지막 원소를 제거하고 제거한 원소를 리턴
  • shift() : 배열의 첫 요소를 제거하고 제거한 원소를 리턴, pop()과 위치가 반대
  • unshift(value) : 배열의 가장 앞에 원소를 추가하고, 추가 후 배열의 길이 리턴
  • concat(array, array2) : 왼쪽기준으로 배열을 합친배열 생성 : let newarray = array.concat(array2, ..)
    array.concat("value") :배열 값 뒤쪽으로 값 추가
  • splice() : 끼워 넣기 array.splice(0 (인덱스 접근 값),1(삭제할 갯수) ,value, value)
    value 값을 안넣으면 삭제도 가능하다
  • slice(index Num) : index 번호로 부터 잘라서 새로운 배열 생성 : let newarray = arrat.slice(n)
    (1, 3 <- 3-1)
