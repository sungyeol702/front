배열의 순환할 수 있는 구조를 이용하여 특정 기능을 할 수 있는 메서드가 제공됨

  • array.reduec(callback function(total, value, index, self), [init value//옵션 들어올수도 안들어올수도있음]) : 내부연산의 누적값을 리턴
  • array.reduecRigth(reduce메서드와 동일) : 배열의 값을 우측에서 좌측방향으로 대입
  • array.every(callbcak function(value, index, self)) : 콜백함수의 리턴조건에 모든 배열의값이 만족하면 true를 리턴 그렇지 않으면 false를 리턴
  • array.some(callbcak function(value, index, self)) : 콜백함수의 리턴조건에 만족하는 배열값이 존재하면true를 리턴 그렇지 않으면 false를 리턴
  • array.indexOf("str value", [start index]) : 배열의 값중 해당하는 문자열이 있는 배열의 index값을 리턴함 : 문자의 indexOf() 메서드와 비슷
  • array.lastIndexOf("str value", [start index]) : 배열의 값중 해당하는 문자열이 있는 배열의 마지막 index값을 리턴함 : 문자의 indexOf() 메서드와 비슷
  • array.find(callbcak function(value, index, self)) : 콜백함수의 리턴조건에 만족하는 첫번째 배열원소를 리턴함 : 검색조건 및 값 리턴이 쉬운편이여서 자주 사용함
  • array.from('sequence architecture가 적용된 객체') : 순서가 있거나 순환구조를 갖고 있는 객체를 배열로 만들 수 있다
  • array.keys() : 배열이 사용중인 key(실별자, 즉 index)를 검색하여 새로운 배열로 리턴 => 객체에는 적용이 안됨
  • array.entries() : 배열의 index:value의 pair구조의 값을 만들어 새로운 배열에 담아 리턴 => 객체에는 적용이 안됨
  • array.includes("찾고 싶은 값") : 배열에 찾는 값이 존재하면 true 리턴
